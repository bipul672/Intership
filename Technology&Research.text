## Q1. What is the technology in data analyst

Technology in data analytics involves the tools, programming languages,
and platforms used to clean, process, and transform raw data into actionable
insights. These technologies are categorized into distinct layers of the data 
ecosystem, ranging from databases to visualization software.

# These technologies are organized into different layers of the data ecosystem:

Data Storage: Databases such as MySQL, PostgreSQL, and SQL Server store structured data.
Data Processing: Tools like Python, Pandas, and NumPy are used for data cleaning and transformation.
Data Analysis: SQL, Python, and statistical techniques help identify patterns and trends.
Data Visualization: Power BI, Tableau, and Excel convert data into interactive dashboards and reports.
Machine Learning: Libraries such as Scikit-Learn and TensorFlow are used to build predictive models.
Version Control & Collaboration: Git and GitHub help manage code and collaborate on projects.

## Q3. Required Skill Set for a Data Analyst

 1. Technical Skills

 **SQL** – Querying and managing data from databases.
 **Python** – Data cleaning, analysis, and automation.
 **Excel** – Data manipulation, Pivot Tables, VLOOKUP/XLOOKUP, Charts.
 **Power BI / Tableau** – Data visualization and dashboard creation.
 **Statistics** – Mean, median, standard deviation, hypothesis testing.
 **Data Cleaning & Transformation** – Handling missing values and data quality issues.

## Q5. Future Scope of Data Analytics

The future of Data Analytics is highly promising as organizations increasingly rely on data-driven decision-making. With the rapid growth of digital technologies, the demand for skilled data professionals continues to rise across industries.

Key Future Trends
Artificial Intelligence (AI) and Machine Learning Integration – Analytics is becoming more predictive and automated.
Big Data Analytics – Companies are processing larger volumes of data than ever before.
Cloud-Based Analytics – Platforms like AWS, Azure, and Google Cloud are transforming how data is stored and analyzed.
Real-Time Analytics – Businesses use live data for faster decision-making.
Data-Driven Business Strategies – Organizations increasingly depend on analytics to improve performance and competitiveness.
